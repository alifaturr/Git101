Pada materi Branch ini dijelasklan tentang apa itu Branch serta fungsi dan kegunaannya.
Pembahasannya meliputi perintah-perintah git yang berkaitan dengan Branch itu sendiri,
Seperti : Git Branch => untuk melihat semua Branch, Git Branch nama_branch => untuk membuat Branch baru, dan lain-lain.
Kemudian dijelaskan tentang apa itu conflict pada Branch dan cara mengatasinya.
Pada akhir sesi, diberikan sebuah tugas untuk membuat Branch baru pada repository, melakukan
Checkout, Commit dan Merge.
