Pada materi GUI ini dijelasklan tentang software berbasis GUI yang digunakan untuk
mempermudah dalam mengoperasikan Git. Software yang digunkan pada materi ini yaitu Sourcetree.
Secara keseluruhan, isi dari materi ini bagaimana cara mengoperasikan Git menggunakan
software Sourcetree yang mana tidak jauh berbeda secara fungsi dengan Git berbasis CLI,
namun dengan menggunakan Sourcetree pengoperasian Git menjadi lebih mudah dan cepat, dimana
akan sangat membantu dalam pengembangan suatu program.
